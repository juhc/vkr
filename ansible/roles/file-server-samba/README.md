# README для Ansible роли file-server-samba

Роль для настройки файлового сервера Samba с открытыми шарами и слабыми паролями.

## Требования

- Ubuntu 20.04 или совместимая система
- Root или sudo доступ

## Использование

```yaml
- hosts: file-servers
  roles:
    - role: file-server-samba
```

## Переменные

Нет дополнительных переменных.

## Результат

После применения роли будет настроен:
- Samba с открытыми шарами без аутентификации
- SMBv1 включен (уязвим к EternalBlue)
- Пользователь с слабым паролем
- Чувствительные данные в открытом доступе

## Тестирование

```bash
# Подключение к открытому шару
smbclient //<server-ip>/public-share -N

# Сканирование уязвимостей
nmap --script smb-vuln-* <server-ip>
```

## Безопасность

⚠️ **ВНИМАНИЕ**: Эта роль создает намеренные уязвимости для обучения!
Используйте только в изолированной среде!
