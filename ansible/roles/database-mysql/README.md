# README для Ansible роли database-mysql

Роль для настройки MySQL сервера со слабыми паролями и открытым доступом.

## Требования

- Ubuntu 20.04 или совместимая система
- Root или sudo доступ

## Использование

```yaml
- hosts: database-servers
  roles:
    - role: database-mysql
```

## Переменные

Нет дополнительных переменных.

## Результат

После применения роли будет настроен:
- MySQL с паролем root: root123
- Удаленный доступ root разрешен
- Тестовая база данных с чувствительными данными
- Отсутствие шифрования соединений

## Тестирование

```bash
# Подключение к базе данных
mysql -h <server-ip> -u root -proot123

# Использование базы данных
USE sensitive_data;
SELECT * FROM users;
SELECT * FROM financial_data;
```

## Безопасность

⚠️ **ВНИМАНИЕ**: Эта роль создает намеренные уязвимости для обучения!
Используйте только в изолированной среде!
