# Документация проекта киберучений

Добро пожаловать в документацию платформы киберучений по защите операционных систем.

## Структура документации

### Техническая документация

- **[Детали используемых технологий](TECHNOLOGIES.md)** — Подробное описание технологий проекта:
  - Proxmox VE / libvirt/KVM
  - Terraform
  - Ansible
  - Bash и Python скрипты
  - Интеграция компонентов

### Документация по безопасности

- **[Критические ошибки безопасности](SECURITY_MISTAKES.md)** — Документация по наиболее распространенным ошибкам настройки безопасности:
  - Аутентификация и авторизация
  - Сетевая безопасность
  - Конфигурация сервисов

  - Управление учетными данными
  - Логирование и мониторинг
  - Файловая система и права доступа
  - Встроенные уязвимости ОС

## Быстрый доступ

### Для администраторов

1. Начните с изучения [технологий проекта](TECHNOLOGIES.md)
2. Изучите [ошибки безопасности](SECURITY_MISTAKES.md) для понимания уязвимостей
3. Ознакомьтесь с основным [README](../README.md) проекта

### Для преподавателей

1. Изучите [ошибки безопасности](SECURITY_MISTAKES.md) для понимания учебных сценариев
2. Ознакомьтесь со [структурой сценариев](../scenarios-structure.md)
3. Изучите [цели обучения](../scenarios/scenario-office-organization/objectives.md)

### Для студентов

1. Ознакомьтесь с [ошибками безопасности](SECURITY_MISTAKES.md) для понимания изучаемых уязвимостей
2. Изучите [быстрое руководство](../scenarios/scenario-office-organization/QUICKSTART.md)
3. Следуйте инструкциям в сценариях
