```

├── Ubuntu Server (192.168.100.10)

    └── Клиентские уязвимости Windows
```

## Слайд 8: Реализация

### Автоматизация развертывания

**Terraform:**
- Автоматическое создание Ubuntu Server в Proxmox
- Настройка сети через cloud-init
- Интеграция SSH ключей

**Ansible:**
- Playbook: **1289 строк кода**
- Автоматическая настройка всех уязвимостей
- Поддержка Linux и Windows

**Скрипт deploy.sh:**
- Полный цикл развертывания
- Проверка зависимостей
- Проверка сетевой связности
- Интерактивное управление

**Время развертывания:** 35-50 минут для полной инфраструктуры
